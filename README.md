# ClassBoard --- Student Leaderboard System

  

**HTML · CSS · JavaScript · localStorage**

  

A lightweight browser-based leaderboard for instructors to motivate

students, award points, and track progress --- **no setup required**.

  

The app runs fully in the browser and stores all data in

**localStorage** (no backend, no database, no internet needed).

[View Demo](https://tokhy1.github.io/Student-Leaderboard-System/)  

------------------------------------------------------------------------

  

# Overview

  

ClassBoard helps instructors:

  

-   Manage multiple classes

-   Add students and award points

-   Track rankings and monthly performance

-   Export results to CSV

  

Just open the HTML file and start using it.

# Getting Started

  

No installation required.

  

1.  Download the project files

2.  Place them in the same folder

3.  Open **index.html** in any browser

  

The app loads with demo data so you can explore immediately.

# Core Features

  

## Class Management

  

-   Create unlimited classes

-   Switch between classes instantly

-   Each class has its own leaderboard and history

-   Delete classes with confirmation

  

## Leaderboard

  

-   Automatic ranking by points

-   Gold, silver, bronze for top 3

-   Progress bars relative to top scorer

-   Quick button to award points

-   Stats overview (students, points, leader)

  

## Points System

  

-   Award or deduct points

-   Quick presets: +5, +10, +25, +50

-   Optional reason for every award

-   Instant leaderboard updates

  

## History

  

-   Full log of all point activity

-   Search by student or reason

-   Color indicators for positive/negative points

  

## Monthly Rankings

  

-   Separate ranking for the current month

-   Top 3 podium view

-   Automatically resets every month

  

## Student Profiles

  

-   Total & monthly points

-   Award history

-   Current rank

-   Remove student option

  

## Export

  

Export class data to CSV for Excel or Google Sheets.

  

Includes: - Student name

- Total points

- Monthly points

- Number of awards

# Data Storage

  

All data is stored in **localStorage**.

  

Important notes: - Data stays on the same browser/device only

- Clearing browser data removes everything

- Data is not synced or backed up

  

Tip: Export CSV regularly as a backup.
